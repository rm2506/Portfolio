# Personal Portfolio

A professional portfolio website built using **React** and **Vite**, deployed on **Vercel**.  
The site provides a structured overview of professional experience, technical projects, skills, certifications, and academic background.

---

## Technology Stack

**Frontend**
- React (functional components, hooks)
- Vite development and build tooling
- Custom CSS-in-JS styling

**Deployment**
- Vercel (GitHub-integrated CI/CD)
- Path rewrites and routing via `vercel.json`

---

## Features

### Projects
- Interactive project gallery with hover highlighting and focus effects  
- Detailed project modals including descriptions, metrics, tech stack, links, and downloadable files  
- Responsive grid layout optimized for desktop and mobile

### Experience & Education
- Structured experience entries with metrics, achievements, and roles  
- Glass-style UI sections for education, skills, and profile details  
- Chip-based skill taxonomy across programming, frameworks, cloud, analytics, and software engineering practices

### Certifications
- Credential cards with organization branding  
- Hover-enhanced “Show credential” buttons for improved interactivity  
- Support for external certificate verification links

### Performance
- Fast development cycles using Vite’s HMR  
- Lightweight bundle for fast load times  
- Fully responsive design across screen sizes
